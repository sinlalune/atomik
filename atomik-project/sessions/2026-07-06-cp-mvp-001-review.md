# CP-MVP-001 review and close (2026-07-06)

## Decision

Owner decision (explicit, recorded from the working session): the S12
two-week block gate is REFORMULATED into a continuous practice, and the
path closes today. CP-MVP-002 (capture sources + local speech baseline,
M3) opens as the active path.

## Rationale

- The gate's first genuine experiential finding arrived before the gate
  started: **M2 without sources and without a real provider has thin
  daily-use value** — "j'ai rien à tester de compliqué, j'ai même pas
  l'IA ni les sources". Two weeks against that surface would test the
  chassis, not the product.
- The gate's *purpose* — friction-driven prioritization from real use —
  was already delivering: on day one the owner reported, and the same day
  saw fixed, 10+ real frictions (unreadable creation inputs, tree
  collapse + panel toggle, CSS selector leaks ×2, soft-break rendering,
  AI accept semantics ×2, new-note visibility/path expectations, AI panel
  resize/dock), plus two design signals recorded in brainstorm/ (tab
  groups per pane as a loved differentiator; evidence-beyond-selection
  instinct).
- M3 capture is precisely what makes daily use real: the physical world
  (handwritten notes, whiteboards, book pages) becomes sources.

## What the gate becomes

A standing practice across all paths, not a step:

```text
the owner uses atomik on the live repository during development
frictions and delights are recorded as project files (brainstorm/,
  sessions/, questions later) — promoted from chat when needed (17)
the agent fixes frictions as micro-units, same-day where possible
each path's review weighs this experiential record, not tests alone
```
